2. Description of Service

Albatross is a social fantasy golf application that allows users to create and participate in private golf pools with friends. Users form groups, select golfers for upcoming PGA tournaments, and compete based on real-world player performance. The App offers three game modes — Tiered Pick, Salary Cap, and One and Done — each designed for private groups of 5 to 100 people.

Albatross is designed for private friend groups where participants know each other. It is not a daily fantasy sports (DFS) platform, sportsbook, or wagering service.
